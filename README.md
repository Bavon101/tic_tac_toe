## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites
- Flutter setup in your local machine


### Setup

- Clone the GitHub Repository
- Go to the Project Directory
- Run ```flutter pub get```
- Run ```flutterrun``` to build the app
